  1. Init your local repository:

    git init

  2. Link your local repository to the remote origin:

    git remote add origin git@example.com:example/test.git

  3. Add your changes to the stage:

    git add .

  4. Add your commit:

    git commit -m "Put name for commit"

  5. Push your commit to remote source:

    git push -u origin master
